# Структура сервера

## Описание

Создать структуру сервера можно исходя из следующих данных:
  1. Строка запроса - GET / HTTP/1.1
    * <Methods>
    * <Urls>
    * <Versions>

  2. Заголовки запроса
    * <Host>
    * some header 2

## Возможные варианты

  * <Host> - роутинг по хосту
  * <Methods> - роутинг по методам
